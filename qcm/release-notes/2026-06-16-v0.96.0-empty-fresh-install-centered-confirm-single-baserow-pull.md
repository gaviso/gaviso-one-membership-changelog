Cleaner first-run and a consolidated Baserow pull.

### Changed
- **Fresh install starts empty** — no default categories (the Build / Engage / Retain seed is gone), and no service groups or line items. Existing sites are unaffected.
- **Pull from Baserow** — the replace-mode confirmation now uses the plugin's centered in-page modal (with a red Confirm button) instead of the native browser dialog.

### Removed
- The per-card "Pull from Baserow" buttons on the Line items, Categories, and Service groups cards. Pulling is consolidated into the single card in **Settings → Data & tables** (select tables + Add / Replace mode). The now-unused per-card handlers, admin-post registrations, and result-notice branches were removed as well.

See CHANGELOG.md for the complete list.
