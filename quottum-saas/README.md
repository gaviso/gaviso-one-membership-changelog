# Quottum SaaS changelog

Self-hosted Openchangelog instance for the **Quottum SaaS** (the Next.js + Supabase app at `app.quottum.io`).

- **Site:** https://changelog.quottum.io
- **Source repo:** [`gaviso/quottum-saas`](https://github.com/gaviso/quottum-saas)
- **Coolify base directory:** `/quottum-saas`

Release entries live in `release-notes/`. They are produced by a GitHub Release on `gaviso/quottum-saas` → n8n webhook → an entry committed here → Coolify redeploys.

The **WordPress plugin (Lite)** changelog lives in [`quottumwp/`](../quottumwp/) and serves at https://changelogwp.quottum.io.
