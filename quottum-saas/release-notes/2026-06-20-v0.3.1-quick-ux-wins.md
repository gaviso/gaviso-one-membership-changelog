Three small features grouped into one quick-shipping release. No schema changes, no breaking changes — drop-in.

### Global search

A working search pill in the topbar:

- Grouped results across **clients, quotes, contracts** (5 per group).
- Debounced 200 ms; **⌘K / Ctrl+K** focuses; **↑/↓/Enter** to navigate.
- Backed by a new ` + "`/api/search?q=`" + ` endpoint that respects org scoping.

### Webhook delivery log

The ` + "`event_log`" + ` table has been populated by the renewal sweep all along — now it has a UI. New **Recent deliveries** section on Settings → Webhook:

- Filter by channel (Webhook / Slack) and status (OK / Errors).
- Expandable rows show the **payload sent** and the **response received**.
- Paginated 25/page.

### Service Group library

The mid-level taxonomy that sits between categories (phases) and line items, replacing the ComingSoon stub. Settings → Service Group library:

- Inline create + edit, grouped by parent category.
- Palette colour picker.
- Line-item usage count per group; delete confirms when detaching items.

### Misc

- No migrations. No env-var changes. No breaking type changes.
