Closes the remaining deferred findings from the v0.7.215 audit sweep.

- **S3:** catalog mutations (categories / service-groups / line-items / coupons) are now `isElevated`-gated — plain members can no longer edit the shared catalog presets (they keep read access for quote-building).
- **S5:** invitation acceptance re-runs the `team_members` feature + `seats` limit gates and refuses (409) when a workspace is at/over its seat cap (closes the post-downgrade / race overflow).
- **S6:** the beta email-confirm TOS clickwrap is reconciled to `user_id`, so confirmed beta users are no longer wrongly shown the re-consent modal.
- **Operator perf/cleanup:** `migrate-subscribers` now lists by stale price id (O(affected) not O(account)); `sync-all-stripe` calls a shared lib fn instead of self-fetching over HTTP; removed a dead constant; operator-authored welcome-popup HTML is now sanitized before render.
