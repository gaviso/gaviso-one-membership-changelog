## Beta program polish
- **Tester emails** — beta testers now get a branded **welcome email** when they're added to the program, and a **"feature unlocked"** note whenever they're granted a new beta feature. Both send from Quottum's own domain (no more silent enrollment).
- **Smoother feature gating** — operators get **instant self-test** access to any beta feature they create, plus tidier server + UI gating so beta features flip on cleanly for the right testers.
