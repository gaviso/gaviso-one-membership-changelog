## Beta tester program (new)
- Operators can flag specific users as **beta testers** and grant each one individual **beta features** (a per-tester catalog, separate from plans and feature flags).
- Beta testers get a **"Submit feedback"** button in the dashboard sidebar — a quick form (Bug / Idea / Other + message + optional screenshot) that auto-captures their workspace and current page.
- New **Beta testers** section in the operator console: add testers (with autocomplete from existing users), toggle features per tester, and triage incoming feedback (status workflow + internal notes, with the screenshot and context attached).

## Plan tiering fix
- Reconciled team tiering: **team features now follow the seat count** — Pro (5 seats) includes team members & roles, matching its seat allowance. Free stays solo.

All operator actions remain audit-logged.
